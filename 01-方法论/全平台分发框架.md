# 全平台分发框架

> 一篇文章→12平台·每条改什么·什么时候发·怎么变现

---

## 一、核心原则

```
一篇深度文（公众号/GitHub·2000字）
    │
    ├── 国内9平台 ← 改长度·改语气·不改核心洞察
    │   变现：微信私域 S-9595A → 课程/培训
    │
    └── 海外4平台 ← 改语言·改文化语境·不改核心洞察
        变现：LinkedIn DM → GEO出海·企业培训
```

**三个不改**：核心洞察不改、真实故事不改、金句不改
**三个必改**：长度按平台改、语气按平台改、钩子按平台改

## 三、一篇深度文→12平台改编对照

### 原文示例："我花了三十年才学会——不是所有问题都需要解决"

| 平台 | 怎么改 | 第一句话 |
|------|--------|---------|
| 公众号 | 原文·2000字 | "我以前特别爱分析。女朋友生气了——我要找到根本原因……" |
| 知乎 | 改回答体·800字·加问题 | 问题"你从哪一刻开始不再焦虑了？" 回答"我前半辈子最大的错误……" |
| 小红书 | IP+国学哲学+个人成长·图文·300字·加封面 | "国学不是背古文——是用两千年前的算法解今天的问题。我做了600篇这样的翻译……" |
| 抖音 | 改口播·60秒·加钩子 | "我告诉你。我前半辈子犯的最大的错——是总想把所有事都彻底搞定。" |
| B站 | 改视频大纲·20分钟·分6段 | "开场：我以前是个完美控（3分钟·讲故事）" |
| 微博 | 改一句话·140字 | "我花了三十年才学会：人生绝大多数难题不是用来解决的，是用来承受的。" |
| 掘金 | 改技术角度·500字 | "做了十年程序员，我才学会不要追求完美架构。" |
| X | 改英文Thread·7条 | "1/ I spent the first 30 years trying to solve everything." |
| LinkedIn | 改英文长文·1000词 | "The Most Expensive Lesson of My 20s: You Can't Optimize Everything" |
