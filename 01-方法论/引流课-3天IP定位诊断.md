# 🎁 ¥39.9 引流课：3天找到你的IP定位

> 大多数IP死在第一步：不知道自己该做什么。3天，帮你找到那个"只有你能做"的定位。

---

## 课程结构

### 第1天：找到你的认知差

**核心问题**：你知道但别人不知道的东西是什么？

你做了十年餐饮。你觉得"利润在供应链不在翻台率"是常识。但对刚开店的人来说——这是值钱的知识。

**作业**：列出你行业内"你以为大家都知道但其实只有你知道"的3件事。

---

### 第2天：定位三层结构

| 层 | 回答 | 你的答案 |
|------|------|---------|
| 功能层 | 你解决什么问题？ | |
| 身份层 | 你的客户是谁？ | |
| 使命层 | 你为什么必须做这件事？ | |

大多数人停在功能层——"我教人做短视频"。功能层可以被替代、被比价。身份层和使命层才有溢价。

**作业**：填完上面的三行——这是你IP定位的第一版。

---

### 第3天：避开四个致命错误

1. **太宽**："我帮人成长"——谁？什么成长？怎么帮？
2. **太窄**："我帮北京30-35岁互联网男优化周报排版"——人群<100人
3. **太像别人**："我是XX领域的XX"——永远是第二个
4. **没有付费场景**："我帮人提升认知"——怎么付钱？

**作业**：检查你的定位有没有踩这四个坑。
