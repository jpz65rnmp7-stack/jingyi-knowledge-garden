# 马修·沃克——"睡眠不足正在杀死你——不是比喻——是生理上"

2017年。UC伯克利神经科学教授马修·沃克出版了《我们为什么睡觉》——被比尔·盖茨推荐为"最好的关于睡眠的书"。沃克说：**"自然界中——没有一种动物不睡觉。睡眠不是奢侈品——是生命维持系统。"**

---

## 睡眠不足的硬后果

| 如果每晚少于6小时 | 发生什么 |
|--------------|---------|
| 免疫细胞攻击能力 | 下降70% |
| 心血管疾病风险 | 显著上升 |
| 睾酮水平 | 等同于老十年 |
| 情绪控制——杏仁核——大脑威胁探测器——上调 | 易怒——焦虑——小刺激大反应 |
| 海马体记忆巩固 | 没存——昨天学的东西=无效 |

**沃克说——"小于7小时的睡眠——第二天——你的表现——即使在你自己感觉不到任何不同的时候——已经下降了。"你只是没有意识到——因为你大脑的"自评系统"本身——也在睡眠不足时——失效了。**

## 怎么做

| 不要 | 要 |
|------|------|
| 周末补觉 | 每天同一时间起——包括周末 |
| 睡前看屏幕 | 睡前一小时无屏 |
| 傍晚喝咖啡因 | 下午2点后不喝咖啡 |
| 躺床上焦虑"睡不着" | 如果20分钟没睡着——起来——去另一个房间——看无刺激纸书——困了回来 |
