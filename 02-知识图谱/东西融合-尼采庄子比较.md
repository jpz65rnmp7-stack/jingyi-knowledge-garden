# 尼采和庄子在同一个悬崖边站着——一个喊"上帝死了"，一个笑"天地一指也"

两千年隔开。一个德国人一个中国人。一个说"上帝死了"把整个西方价值体系炸掉了。一个说"天地一指也"把所有人类的认真当成笑话。但他们是两个描述同一个真相的人——只用不同的语言。

---

## 尼采：你必须自己创造价值

"上帝死了"不是说上帝不存在。是说——曾经所有的"应该"——宗教给的道德、社会给的规范、传统给的价值观——不再有效了。你不能再依靠外部权威告诉你该怎么活。你必须自己创造自己的价值。

**"成为你自己"——尼采不是让你"发现你是谁"。是让你"创造你是谁"。你不是找到的。你是做出来的。**

## 庄子：所有的"应该"都是人类自己的发明

"天下莫大于秋毫之末，而泰山为小；莫寿于殇子，而彭祖为夭"——视角不同，大小、长短、是非——全是相对的。泰山在宇宙里=一粒尘埃。刚出生就死去的婴儿——如果跟朝生暮死的虫子比=已是长寿。

**庄子说：你认真对待的"大事"——从另一个角度看——可能是小事。你焦虑的"必须"——是你自己造出来的。**

| 尼采 | 庄子 |
|------|------|
| "成为你自己"——创造价值 | "逍遥游"——不执着于任何单一价值 |
| 超人：超越"应该"的人 | 真人：随自然流动的人 |
| 强力意志——意志驱动 | 无为——顺应道 |

**两个人都到了同一个地方：别让别人告诉你该怎么活。尼采说"自己创造价值"。庄子说"不执着任何价值"。看似矛盾——但他们在同一个山顶。一个登山者绕了不同的路。**
