# 人物参考图与首帧图提示词 001-002

## 使用顺序

1. 先生成统一人物参考图。
2. 选择最稳定、最普通、最真实的一张。
3. 后续所有人物分镜都上传这张图作为参考图。
4. 每条人物分镜提示词开头加：`参考图：[图片链接]`
5. 空镜头不需要人物参考图，直接用分镜提示词生成。

## A. 统一人物参考图提示词

用途：给 001、002 两篇都作为主角参考图，保证人物一致。

```text
9:16竖屏，真实电影感人物肖像，一位35-45岁的普通中国成年人，可以是男性或女性，长相普通，不是明星脸，不是网红脸，穿深灰色或黑色简单外套/居家卫衣，发型自然，脸上有轻微疲惫感，眼神克制，有生活压力但没有夸张表情，像一个撑了很久但还在努力生活的人。

背景是普通家庭客厅或窗边，环境简洁，低饱和暖灰暗蓝色调，柔和侧光，真实摄影质感，浅景深，皮肤纹理自然，人物半身中景，正面或微侧脸，双手自然放松。

不要豪华感，不要精英感，不要过度美颜，不要卡通，不要赛博朋克，不要水印，不要logo，不要文字，不要夸张哭泣，不要戏剧化表演。
```

建议生成 5-8 张，选择标准：

- 脸稳定，不畸形
- 表情克制，不演
- 年龄像 35-45 岁
- 服装简单，后续镜头容易延续
- 背景普通，不抢戏

## B. 001《你不是不努力，你是太怕自己没用》封面/首帧图提示词

用途：做抖音/视频号封面、视频开头首帧、即梦图生视频起始画面。

```text
参考图：[人物参考图链接]

9:16竖屏，电影感现实主义，深夜普通家庭客厅，同一位35-45岁普通中国成年人坐在沙发边，身体微微前倾，手里攥着手机，低头沉默，像刚刚意识到自己不是不努力，而是太怕自己没用。

桌上有半杯凉水、钥匙、银行卡、没有可读文字的账单纸、孩子作业本和药盒，屋里只开一盏暖色小台灯，窗外是暗蓝色城市夜景。人物表情疲惫、克制、心里有事，但不要哭，不要夸张。

画面下方留出字幕空间，整体低饱和暖灰暗蓝色调，真实摄影质感，浅景深，细节真实，有成年人生活压力感。无文字，无水印，无logo，不要豪华装修，不要广告片质感。
```

封面标题建议：

```text
你不是没用
你只是太累了
```

## C. 002《你不是脾气差，你是太久没人心疼》封面/首帧图提示词

用途：做抖音/视频号封面、视频开头首帧、即梦图生视频起始画面。

```text
参考图：[人物参考图链接]

9:16竖屏，电影感现实主义，深夜普通家庭客厅或餐桌旁，同一位35-45岁普通中国成年人坐在桌边，刚刚发完火后安静下来，低头看着桌面，表情后悔、疲惫、委屈，但没有哭泣，没有夸张表演。

远处孩子房门半开，屋里很安静，桌上有半杯水、作业本、铅笔、纸巾、没有可读文字的纸张。光线一半是暖色台灯，一半是暗蓝夜色，表现“发完火以后家里突然安静”的感觉。

画面下方留出字幕空间，低饱和暖灰暗蓝色调，真实摄影质感，浅景深，生活细节真实。无文字，无水印，无logo，不要卡通，不要戏剧化哭喊，不要豪华装修。
```

封面标题建议：

```text
你不是脾气差
你是太久没人心疼
```

## D. 后续分镜使用方式

人物镜头：

```text
参考图：[人物参考图链接] + 对应分镜头提示词
```

空镜头：

```text
直接使用对应分镜头提示词
```

批量生成：

- 人物参考图生成 5-8 张，选 1 张固定
- 每个分镜生成 3-5 版
- 不要频繁更换参考图
- 同一篇视频里人物服装和年龄必须一致

## E. 质量检查

生成后检查：

- 人物是否像同一个人
- 服装是否稳定
- 年龄是否符合 35-45 岁
- 表情是否真实克制
- 是否出现文字、水印、畸形手脸
- 画面是否保留字幕空间
