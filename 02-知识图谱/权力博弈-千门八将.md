# "千门八将"——不是教你骗人——是让你看清骗子怎么运作

千门——中国最古老的"骗子职业共同体"。他们总结出——**任何骗局都需要八种角色的配合——缺一种——局就不完整。** 你不知道这些角色——你永远看不出"为什么我会上当"。

---

## 八将分工

| 将 | 角色 | 功能 |
|------|------|------|
| 正将 | 主脑 | 设计骗局的人——搭"舞台" |
| 提将 | 策划/沟通 | 拉人来——"舞台上的演员招呼观众坐下" |
| 反将 | 激将·引诱 | "不买=土"——用"社会比较"让你失去冷静 |
| 脱将 | 善后·洗白 | 骗完了——让你投诉无门——他已经消失了 |
| 火将 | 面子·气场 | 假装有权有势——用"伪权威"让你信任 |
| 除将 | 攻防·分散注意 | "这位顾客请等一下——您要的东西在那边"——你一转身——另一个"同伴"让你放松 |
| 谣将 | 信息污染 | 散布谣言——"内部消息——马上翻倍——信不信由你" |
| 风将 | 监控·望风 | 全程监控——同行来干扰时——紧急解散 |

**你遇到一个骗局——你只看到了一个人——但实际上——他背后有八个角色在协调运行。你看不到他们——但你感觉"那个人好像很可靠"——其实那个"可靠感"——是八个人合力制造出来的"氛围"。**
