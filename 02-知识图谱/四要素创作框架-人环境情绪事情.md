# 四要素创作框架：人·环境·情绪·事情

## 为什么是这四个？

任何人的任何困惑，都发生在一个四维空间里：

```
          人 (Human)
         /  \
        /    \
       /  困  \
      /   惑   \
     /          \
   情绪 ——————— 事情
 (Emotion)    (Action)
      \        /
       \      /
        \    /
         \  /
         环境 (Context)
```

- **人**：谁在经历？他的性格、欲望、恐惧、关系、底层假设是什么？
- **环境**：在什么系统里发生？时机、规则、资源、约束条件是什么？
- **情绪**：身体在经历什么感受？这个感受在传递什么信号？它怎么影响判断？
- **事情**：本质是什么？能做什么？不能做什么？最坏结果能不能承受？

**四个维度缺一个，认知就不完整：**
- 只看人 → 自恋式反思，看不到系统和情绪
- 只看环境 → 宿命论，看不到能动性
- 只看情绪 → 沉溺式宣泄，看不到认知和行动
- 只看事情 → 莽撞式执行，看不到人和情绪的信号

## 每条脚本的四要素检查清单

脚本写完后，逐项检查：

### 人（至少覆盖一项）
- [ ] 人性规律：为什么人会这样想/这样做？
- [ ] 自我认知：读者能看到自己什么盲区？
- [ ] 关系本质：人与人互动背后的底层逻辑？

### 环境（至少覆盖一项）
- [ ] 外部条件：时代/行业/家庭/社会等不可控因素
- [ ] 系统规则：明规则+暗规则+权力结构
- [ ] 位置时机：读者在什么位置？现在是不是对的时机？

### 情绪（至少覆盖一项）
- [ ] 情绪命名：帮读者说清楚"我到底在感受什么"
- [ ] 情绪来源：这个情绪从哪里来？身体在传递什么信号？
- [ ] 情绪转化：从被情绪控制到与情绪共处，给方法

### 事情（至少覆盖一项）
- [ ] 问题本质：剥掉情绪和假设，这件事到底是什么？
- [ ] 可控动作：读者能做的具体动作（不要只讲道理）
- [ ] 结果预判：做了会怎样、不做会怎样、最坏能不能承受

## 示例：同一条选题，四个切入角度

选题："为什么我越努力越焦虑？"

### 从【人】切入
你在跟谁比？你脑子里那个"裁判"是谁装的？你的底层假设是"我必须比别人强才有价值"——这个假设从哪来的？

### 从【环境】切入
你所在的行业在上升还是下沉？你的努力是在一个奖励努力的系统里，还是在一个只奖励结果的系统里？你手里的牌和别人一样吗？

### 从【情绪】切入
焦虑首先是身体感受——心慌、失眠、紧绷。你的身体在用焦虑告诉你什么？是"该停了"还是"该变了"还是"你太累了"？

### 从【事情】切入
你努力的具体动作是什么？这些动作有复利吗？能不能积累？如果五年后再看，今天的努力是在铺路还是在原地踏步？

**一条完整的脚本，应该至少覆盖三个维度，其中一个是主要切入角度。**

## 商业价值

此框架本身就是可商业化的方法论：
- 可做成"四要素人生诊断"一对一咨询产品
- 可做成课程/训练营的思维模型模块
- 可做成检查清单/工具卡作为私域钩子
