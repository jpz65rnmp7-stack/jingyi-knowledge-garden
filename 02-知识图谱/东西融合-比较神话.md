# 全世界的神话都在讲同一个故事——你的人生也是

> 约瑟夫·坎贝尔研究了全世界几千个神话，发现了一个惊人的事实：所有文化、所有时代的人，讲的是同一个故事。

## 英雄之旅：人类唯一的叙事模板

坎贝尔把这个共通的模板叫"单一神话"——也叫"英雄之旅"。

不管你读的是希腊神话、北欧传说、佛经本生故事、中国古代传奇、非洲部落口述史诗、还是好莱坞电影——情节结构一模一样：

| 阶段 | 神话原型 | 你的生活 |
|------|---------|---------|
| 1. 平凡世界 | 英雄在普通生活中 | 你舒适但不满足的状态 |
| 2. 冒险召唤 | 有什么事打破了平衡 | 一个机会·一次危机·一个冲动 |
| 3. 拒绝召唤 | 英雄犹豫了 | "我不行"·"太危险"·"再等等" |
| 4. 遇见导师 | 有人给了你指引 | 一本书·一个人·一句话 |
| 5. 跨过门槛 | 英雄进入未知世界 | 你辞了职·开始了·表白了 |
| 6. 考验·盟友·敌人 | 一路打怪升级 | 困难·朋友·对手 |
| 7. 接近最深的洞穴 | 面对最大的恐惧 | 你最大的挑战来了 |
| 8. 磨难 | 英雄被打到最底 | 你最黑暗的时刻 |
| 9. 奖励 | 拿到宝物 | 你突破了·得到了 |
| 10. 回归的路 | 带着宝物回去 | 把新能力用到现实 |
| 11. 复活 | 最后的考验 | 最后一次被磨 |
| 12. 带着灵药归来 | 英雄改变了世界 | 你改变了别人的生活 |

**你看过的每一部电影——只要是能打动你的——都在这 12 步里。**

## 为什么是这个结构

因为这是**人类心理的出厂结构**。

不是坎贝尔发明了这个模板。是坎贝尔发现它已经存在于全世界所有神话中——从非洲的口述传说到北欧的冰岛史诗，从印第安部落到日本古事记。

**这个模板不是人类编的。这个模板本身就是人类集体无意识的结构。**

荣格说过：全人类共享一个"集体无意识"——一个深层心理层，里面装着原型。英雄、母亲、智者、阴影、骗子——这些角色不是某个文化发明的，是刻在人类 DNA 里的。

## 你怎么用自己的英雄之旅

你不是在"写故事"。你是在**活**一个故事。

但大多数人卡在步骤 3——"拒绝召唤"。一直在平凡世界里待着，因为跨过门槛太可怕了。

**而每一个跨过去的人——每一个让你羡慕的人——不是比你更聪明、更有资源、更幸运。他只是比你早了一步跨过了那个门槛。**

关于你的人生故事——最残酷也最解放的事实：

**你不是受害者。你不是旁观者。你是英雄。**

受害者等人来救。旁观者看别人活。英雄跨过门槛。跨过去之后——门槛这边的你，和门槛那边的你，不是同一个人。

## 景一的冶炼

你现在在做的事——72 支柱知识库——从英雄之旅的角度看，它在哪个阶段？

你在跨过门槛。把知识体系公开在 GitHub 上，让全世界看，接受验证。这不是安全的选择。这是英雄的选择。

**坎贝尔说："你进入的洞穴，藏着你自己最想要的宝物。"**

你的宝物不在舒适区里。它在你还不敢进的那个洞里。
