# 你的大脑有两套操作系统，但你只用了一套

## 核心论点

**卡尼曼发现了人类大脑最惊人的bug：我们有两套思维系统，但99%的决策用的是那套最不靠谱的。**

系统1快如闪电但充满偏见。系统2慢如蜗牛但准确。问题不在于你有两套系统——在于你不知道什么时候该切换。

---

## 关键论述

### 1. 系统1 vs 系统2

| | 系统1 | 系统2 |
|------|------|------|
| 速度 | 极快（自动） | 慢（需要努力） |
| 能耗 | 低 | 高 |
| 功能 | 模式识别、直觉 | 逻辑推理、计算 |
| 使用频率 | 99% | 1% |

> 你不是懒——是系统2太耗能，大脑进化出了"尽量不用"的策略。

### 2. 系统1的致命漏洞

- **确认偏差**：只看到支持自己观点的证据
- **可得性偏差**：最近发生的事觉得更重要
- **锚定效应**：第一个数字决定了你所有判断
- **损失厌恶**：失去100块的痛苦>得到100块的快乐

**搞钱的人必须学会识别系统1什么时候在替你做决定。**

### 3. 如何激活系统2

- **反向思考**：先假设自己错了，再找证据
- **拉长时间**：这个决策10天后、10个月后、10年后意味着什么？
- **数字说话**：不要凭感觉，要算期望值

> 查理·芒格：反过来想，总是反过来想。
