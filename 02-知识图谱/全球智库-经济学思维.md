# 经济学不是研究钱的——是研究"人怎么选"的

> 你以为经济学是关于 GDP、通胀、股市。其实经济学是关于一个根本问题：当一个东西不够分的时候，谁得到，谁得不到。

## 十个经济学原理，九个被误读

大多数人提到"经济学"，脑子里蹦出来的是股票、楼市、通货膨胀。

这些是经济学的外包装。撕开包装，经济学只有一个核心问题：**稀缺。**

资源有限。欲望无限。所以必须选择。选择 = 舍弃。舍弃 = 成本。成本 = 你放弃的那个东西，不是你的钱。

**这是经济学给你上的第一课：你花的不是钱，是你放弃的下一个最好的选择。**

## 边际思维：不要看"总量"，看"多一个"

大部分人的思维是：这件事值不值得做？

经济学家的思维是：**多做一个单位，值不值得？**

| 普通人 | 边际思维 |
|--------|---------|
| 值不值得开店？ | 多开一个小时，多赚的和多花的，哪个大？ |
| 值不值得再招一个人？ | 这个人产生的价值，大于他的成本吗？ |
| 值不值得继续这段关系？ | 多投入一天，得到的和失去的，哪个多？ |

**边际思维 = 不看过去（沉没成本），不看总量（太重了），只看"多一个"。**

已经投进去的时间和钱不参与了——沉没了。你只需要问自己：从现在开始，多做一个单位的投入，值不值？

## 激励：人不是按你说的行动，是按"什么奖励他"行动

经济学最残酷的洞察：**不要听人说什么。看什么在奖励他。**

一个人说"我最在乎品质"——但如果他每次被发现偷工减料都没后果，他迟早偷工减料。不是因为他坏。是因为激励结构允许。

一个人说"我忠于公司"——但如果竞对给他涨薪 50%，他不走？不是忠诚。是激励不够大。

**设计任何系统——团队、合作、关系——不是靠"说清楚期待"。是靠"设计什么行为被奖励、什么行为有代价"。**

## 交易成本：为什么有些事你自己做，有些事交给别人

你想吃什么菜，不会自己去种地。因为种地的时间成本远大于叫外卖的钱。

你想学什么技能，不会自己去研究所有资料。因为筛选的成本远大于买一门课的钱。

**所有商业的本质，都是降低交易成本。所有个人的成长，都是识别"什么值得自己花时间、什么应该买别人的时间"。**

## 景一的冶炼

经济学给出了四个终身受用的思维工具：

| 工具 | 问自己 |
|------|--------|
| 机会成本 | 我选这个，放弃了什么？ |
| 边际思维 | 多做一个单位，值不值？ |
| 激励 | 什么在真正奖励我/别人？ |
| 交易成本 | 自己做还是买别人的时间？ |

**经济学不研究钱。经济学研究人。而人——是你这辈子要跟它打一辈子交道的唯一变量。**
