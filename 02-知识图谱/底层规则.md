# 🕸️ 底层规则：暗流规则体系精炼

> 来源：暗流规则 | 维度：中层认知 | 核心命题：游戏规则是什么

---

## 什么是隐学

**显学** = 公开教你的东西（努力就能成功、好人有好报）

**隐学** = 真正运行世界的规则（信息差决定财富差、权力结构决定资源分配）

隐学不是阴谋论。是**权力和信息的真实运作逻辑。**

---

## 八大核心洞察

### 1. 信息封锁
- 最值钱的信息从来不在公开渠道
- 信息差不是技术问题，是**权力结构**
- 有人不想你知道，不是因为你不配知道，是因为你知道了他们就不值钱了
- **行动**：你现在最需要但最难获取的信息是什么？主动去找，不要等它送上门

### 2. 强势文化
- 弱势文化 = 等别人给你答案（等老板、等政策、等机会）
- 强势文化 = 自己制定规则，自己创造答案
- 强势不是凶，是**不把定义自己的权力交给别人**
- **行动**：你今天有哪个决定是在等别人点头？能不能自己决定？

### 3. 人性秘术
- 所有的人性操控都基于三个杠杆：恐惧、贪婪、懒惰
- 操控 = 利用这三个杠杆让你做有利于操控者的事
- 防御 = 意识到自己被触发了哪个杠杆
- **行动**：回想上一次你被说服的经历 → 对方动用了恐惧、贪婪还是懒惰？

### 4. 权力结构
- 任何组织/关系里都有一个隐形权力结构
- 名义上的权力 vs 实际的权力，往往是两个人
- 看谁决定资源分配，不是看谁头衔最大
- **行动**：在你所在的环境里，谁真正决定资源分配？

### 5. 规则是强者写的，但借口是弱者给的
- 强者制定规则 → 弱者遵守规则 → 更弱者抱怨规则
- 与其抱怨不公平，不如**理解规则，利用规则，最终改写规则**
- **行动**：你现在最抱怨的一件事，背后是什么规则？你是在理解它还是抱怨它？

### 6. 认知差 = 阶级差
- 你的收入不会超过你最近的五个朋友的平均值
- 不是因为"圈子决定命运"这种鸡汤 → 是因为**信息源决定认知，认知决定决策**
- **行动**：审视你的信息源，你每天在吃什么样的认知？

### 7. 不把定义权交给别人
- 别人说你不行的底层逻辑：你行了会影响他的利益
- 别人说你可以的底层逻辑：你可以了会给他带来利益
- 别人的评价 = 别人的利益计算
- **行动**：今天有谁的评价在影响你？他的利益和你的利益一致吗？

### 8. 隐学的学习方法
- 不是背书、不是上课
- 是**观察 + 推演 + 验证**
- 观察：谁在做什么、获得了什么
- 推演：背后的规则是什么
- 验证：下次类似情境中验证你的推演
- **行动**：今天观察一个你身边的人际互动，推演背后的利益逻辑

## 底层规则一句话

**游戏规则从来不写在说明书上。看懂规则的人不抱怨，直接玩。看不懂的人一直在问"凭什么"。**
