# 四维破局 · 景一原创认知体系

> 任何人生困局，都可以拆进四个维度。
> 缺一个维度，认知就不完整。

---

## 体系总览

```
        🔭 看局 (看清全局)
       /  \
      /    \
     /  四  \
    /   维   \
   /   破    \
💙 修心 ——— 🎯 成事
(稳住内在)   (拿到结果)
      \      /
       \    /
        \  /
         👤 识人 (读懂人性)
```

---

## 四维详解

### 🔭 看局 — 看清全局，不盲打

> "问题不是你看错了，是你根本没看全。"

**维度内涵：**
- 环境：你在什么系统里？行业上升还是下沉？
- 规则：明规则+暗规则+权力结构
- 时机：现在是不是对的时机？《易经》"时位"——时间不对做了就是灾
- 趋势：这是暂时的波动还是结构性的变化？

**核心问题：** 我以为是我的问题，其实是我在这个环境里、在这个时间点、在这些规则下——谁来做都一样。

**典型应用：** 职业选择、行业判断、商业决策、投资时机

---

### 👤 识人 — 读懂人性，不天真

> "你对人性的失望，不是因为人太坏，是因为你把人想得太简单。"

**维度内涵：**
- 人性规律：趋利避害、价值交换、信息不对称
- 关系本质：利益+情感+边界的动态平衡
- 自我认知：你的底层假设是什么？哪些"应该"在限制你？
- 他人动机：不看他怎么说，看他在利益面前怎么选

**核心问题：** 他不是针对我，他是站在他的位置、他的利益、他的认知里做出的选择。我要做的不是生气，是调整我的位置和边界。

**典型应用：** 人际关系、职场博弈、商业谈判、亲密关系

---

### 💙 修心 — 稳住内在，不内耗

> "情绪不是软弱，是身体在给你发信号。"

**维度内涵：**
- 情绪信号：焦虑/愤怒/羞耻/麻木在告诉你什么？
- 能量管理：像管钱一样管能量，先止漏再蓄力
- 定力训练：十分钟规则、控制二分法、认知预演
- 边界建立：善良带锋芒、解释欲戒断、讨好模式识别

**核心问题：** 情绪上头时不做决定。先接住情绪，再处理事情。一个情绪不稳的人，看到的局、识到的人、做成的事都是变形的。

**典型应用：** 焦虑管理、关系修复、重大决策、自我重建

---

### 🎯 成事 — 拿到结果，不空谈

> "知识不进入行动就是信息。行动没有闭环就是浪费时间。"

**维度内涵：**
- 问题本质：剥掉情绪和假设，这件事到底是什么？
- 可控动作：四层拆事法——事实层→利益层→规则层→趋势层
- 执行系统：小到不可能失败的动作+固定流程+复盘迭代
- 复利积累：今天的动作，一年后还有没有价值？

**核心问题：** 不追求一步到位。追求"今天我做了一个对的动作，明天继续"。普通人的改变不是突然发生的，是一点一点顺起来的。

**典型应用：** 目标管理、习惯养成、职业转型、副业启动

---

## 四维之间的关系

```
看局不清 → 识人必偏 → 修心无根 → 成事必败
看局清晰 → 识人准确 → 修心有据 → 成事可期
```

- **看局**是地基——没看清环境，后面全歪
- **识人**是关键——人不对，再好的局也破不了
- **修心**是桥梁——心不稳，看局和识人的判断全被情绪污染
- **成事**是检验——前三个维度都要落在具体动作上

## 使用方法

**个人诊断：** 遇到任何困惑 → 四个维度各问自己一个问题 → 找到核心卡点 → 对症下动作

**内容创作：** 每条脚本必须覆盖至少三个维度 → 确保"解惑"不偏科

**商业咨询：** 帮客户四维扫描 → 定位问题在哪个维度 → 给出那个维度的具体动作
