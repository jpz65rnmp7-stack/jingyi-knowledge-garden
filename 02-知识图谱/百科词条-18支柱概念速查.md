# 景一知识体系·核心概念速查

> 18支柱48个核心概念，一句话定义。这是你的话语体系。

---

## 核心三支柱

| 概念 | 定义 |
|------|------|
| 钱途心法 | 洞察人性底层需求，设计可持续变现系统 |
| 人情账本 | 社会交换的无形记账——你的善意是存款，你的索取是提款 |
| 转运 | 不是玄学——是行为模式切换引发的反馈机制变化 |
| 执行力 | 不是懒——是对"可能失败"的恐惧转化为"没做"的内疚 |
| 暗流规则 | 社会运行的隐性操作系统——规则只对价值差距不大的人有效 |
| 信息封锁 | 权力结构的基础设施——最隐蔽的封锁是"让你以为你已经知道了" |
| 强势文化 | 不把定义自己的权力交给别人 |
| NPC | 从不审视人生剧本、按默认程序活着的人 |
| 系统降权 | 觉醒时的正常反应——系统排斥异常值 |
| 维度思维 | 从玩家视角看三维剧情 |
