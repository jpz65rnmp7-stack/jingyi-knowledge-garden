# 心流不是"等状态来"——契克森米哈伊发现了九个触发器

九个条件——你满足越多——进入心流的概率越大。不是"今天看运气"。是"今天我满足了几个条件"。条件够=心流会来。不够=你干等。等不来。

---

## 九个心流触发器

| 类别 | 触发器 | 操作 |
|------|--------|------|
| 环境 | 1. 明确目标 | 每一步都知道要做什么 |
| 环境 | 2. 即时反馈 | 每完成一步立刻有反馈 |
| 环境 | 3. 挑战-能力匹配 | 刚好有点难——但够得着 |
| 心理 | 4. 高度专注 | 只做一件事——不做其他 |
| 心理 | 5. 控制感 | 你掌握——不是被掌控 |
| 心理 | 6. 自我意识消失 | 忘了自己——只有手上的事 |
| 心理 | 7. 时间扭曲 | 不知不觉几小时过去了 |
| 社会 | 8. 共同目标 | 团队心流——大家往同一个方向 |
| 创意 | 9. 新奇与探索 | 做一些你以前没做过的事 |

**九个条件——你不需要满足全部。但至少要满足五个以上。三个勉强。两个靠运气。0个=你不可能进心流。**

---

## 心流前清单

开始任何深度工作之前——30秒自查：目标清楚吗——我知道下一步做什么吗。反馈来源清楚吗——做完之后我怎么知道好不好。难度合适吗——太难——降。太简单——升。干扰清除了吗——手机在视线之外。门关着。你觉得"我在控制"——不是被迫——是我选了做这件事。

**清单勾满五个=你不需要"等状态"。状态自己来了。**
