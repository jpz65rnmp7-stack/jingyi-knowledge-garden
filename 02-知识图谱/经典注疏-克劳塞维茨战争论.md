# 克劳塞维茨——"战争是政治通过其他手段的延续"

卡尔·冯·克劳塞维茨。普鲁士将军。死在1831年——书没写完。他的妻子玛丽整理了《战争论》手稿。这本书被所有军事学院尊为圣经。核心一句话："战争是政治通过其他手段的延续。"战争不是为了"赢"。战争是为了实现政治目的——胜和负都是工具——不是终点。

## 关键概念：摩擦和迷雾

| 概念 | 含义 | 商业对应 |
|------|------|---------|
| 摩擦 | 计划里永远不会包含的所有"意外"——下雨——通信断了——人犯错——最简单的任务变困难 | 你的项目——所有微小的日常阻力——加起来让计划慢了十倍 |
| 迷雾 | 你永远不知道全部信息——敌人在哪——多少——什么时候进攻 | 市场同——竞对在做什么——你不会完全知道 |
| 决断点 | 交手最激烈的时刻——谁能更快看清局势——谁的意志更强——谁赢 | |

**克劳塞维茨不是教你"不怕战斗"。他教你——"不要上来就干"。干之前——必须清楚——你的政治目的是什么——军事只是手段。**
