# 最贵的知识不能写在书里——只能"跟人学"

> 你能写的出来的=显性知识=所有人都能学到。你不能写出来的东西——那才是你值钱的地方。

## 两种知识

| 显性知识 | 隐性知识 |
|---------|---------|
| 可以写在书里 | 无法表达 |
| 可以录成课程 | 只能在做的时候被感知 |
| 可以被AI学习 | AI学不会 |
| 任何人都能获取 | 只有跟对人才能吸收 |

**你所有值钱的东西——全在隐性知识里。**

## 为什么隐性知识只能在人跟人之间传播

你不能通过读"游泳教程"学会游泳。

你看一万小时的木工作品——你的手还是不会做。你读了所有"如何成为一个好的CEO"——你还是不会。因为隐性知识不在文字里。隐性知识在一个人面对一个具体情况的瞬间——他做的那个判断里。

**那个判断——他当时可能自己都说不清楚为什么那么做。但那就是他最值钱的地方。**

## 学徒模型：最古老也最有效的学习方式

| 现代教育 | 学徒模型 |
|---------|---------|
| 老师讲→学生记 | 师傅做→徒弟看→徒弟做→师傅改 |
| 知识=抽象概念 | 知识=具体行动 |
| 考试检验 | 产出检验 |
| 六年十万个同龄人 | 三年一个师傅 |

**现代教育量产了"知道的人"。学徒模式培养"能做的人"。前者供过于求。后者永远稀缺。**

## 你怎么用师徒模型

| 学什么 | 找谁 |
|--------|------|
| 写作 | 找一个你佩服的写作者→模仿→请他看你的东西→听他怎么说 |
| 商业判断 | 找一个做过的人→跟他一起做决策→看他在同样场景下怎么想 |
| 你的任何瓶颈 | 找到一个已经过了那个瓶颈的人→离他近一点——物理上或数字上 |

**不需要正式的"拜师仪式"。你只需要靠近那个人——观察、模仿、请他指出你最差的地方——然后只练那个。**

隐性知识不能抄。但可以通过靠近被吸收。像一种气味——你离得够近，你自己也会带上一部分。
