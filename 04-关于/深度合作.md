# 🤝 深度合作

> 不接广告，只做对双方都有价值的深度合作。

## 我能提供什么

### 1. 定制内容
为你的品牌/产品定制跨赛道融合内容：
- 用国学讲你的产品逻辑
- 用心理学做你的用户洞察
- 用思维模型拆你的行业机会

**交付物**：文章、口播脚本、图文卡片、课程大纲

**适合谁**：需要"不一样的视角"来讲自己故事的个人/品牌

### 2. 一对一咨询
针对你的具体问题，从人性底层给你解法：
- 个人IP定位和市场切入
- 搞钱卡点和突破方法
- 认知升级和觉醒路径

**交付物**：60-90分钟深度对话 + 行动清单

**适合谁**：卡在一个瓶颈期，需要有人帮你看清底牌

### 3. 培训/工作坊
面向团队或社群的主题培训：
- "从人性出发"内容创作法
- 跨赛道思维训练
- 用国学搞钱实战工作坊

**交付物**：定制培训方案 + 实操练习 + 后续跟进

**适合谁**：想提升团队认知和创作能力的组织

### 4. 联名内容
两个赛道的人做跨领域的联名内容：
- 你出领域深度，我出跨赛道视角
- 碰撞出单独一方做不出的内容

**交付物**：联合文章、对谈视频、联名课程

**适合谁**：有自己专业壁垒、想破圈的创作者/专家

---

## 合作原则

1. **只做真价值** — 不接硬广，不做违心的推荐
2. **对齐人性** — 合作内容必须符合"从人性出发"的标准
3. **双方增益** — 不能只对一方有利
4. **宁缺毋滥** — 一个月最多接一个深度合作
