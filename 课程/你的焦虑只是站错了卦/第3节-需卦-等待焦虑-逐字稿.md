# 第3节：需卦——那个等结果等到崩溃的人

**卦象**：需 ☵☰（坎上乾下，水在天上）
**焦虑**：投了简历石沉大海、给对方发了消息三天不回、等一个体检结果等到崩溃——等待本身比坏结果更折磨人
**时长**：约18分钟

---

（开场钩子）

你有没有过这种感觉——

你发了一条消息给对方。五分钟过去了，没回。十分钟过去了，没回。半小时过去了，还是没回。

然后你的大脑开始自动播放一部电影。主演是你，主题是"他不回消息的108个原因"。

1分钟版本：他在忙，可能没看手机。
5分钟版本：他看到了，但不想回。
10分钟版本：他不回我，是不是我上次说了什么？
30分钟版本：他可能不想跟我做朋友了。

然后手机亮了。他回了一句：刚才在开会。

（停顿两秒）

你松了一口气。但你有没有算过——你浪费了多长时间？你浪费了整整30分钟，制造了一个完全没有发生的剧情。然后你还会觉得："还好是虚惊一场"——然后下次继续。

这就是等待焦虑。而三千年前，有一卦专门讲这个事。它叫需卦。

（卦象解密）

需卦。坎上乾下。水在天上——云在天上，雨还没下来。

你感觉到了吗？"云在天上，雨还没下来"——这不就是你等待的状态吗？你知道雨会来，但你不知道什么时候来。你看得到云在变厚，但你就是等不到那第一滴雨。

这就是需卦。需，就是等待。

但需卦的卦辞，不是"等待很痛苦你要忍一忍"。它说的是——"需，有孚，光亨，贞吉，利涉大川。"

四个关键词：有孚——有信心。光亨——光在来的路上。贞吉——坚持是吉利的。利涉大川——有利于渡过大河。

翻译成现代话：等待是有意义的。前提是——你在"有孚"地等。有信心地等，不是焦虑地等。

（焦虑连接）

好，这里有一个关键区分。大多数人分不清两种等待。

第一种等待叫"被动等待"——你投了简历，然后每天刷邮箱。你发了消息，然后盯着屏幕。你等体检结果，然后反复搜症状。

这种等待的特征是：**你除了等，什么都没做。** 你的大脑一空转，就开始编剧情。

第二种等待叫"有孚等待"——你投了简历，然后继续优化第二份简历。你发了消息，然后去做你本来要做的事。你等体检结果——你开车去健身房，因为你知道不管结果怎样，身体都需要变好。

这种等待的特征是：**你承认自己在等待，但你手上有事做。** 你的大脑没空编剧情——它忙着做你给它的正事。

（今晚的行动）

练习叫"等中做"。三步。

**第一步：识别——你现在在等什么？** 写下来。

**第二步：区分。** 左列："我只能等的部分"。右列："我现在能做的部分"。

**第三步：从右列挑一件事，现在做。** 不需要是大事。最小的动作就够了。

你一开始做事，焦虑就变小了。不是焦虑自己消失了——是你的大脑没有多余带宽去编剧情了。这就是需卦"利涉大川"的真相——不是等待让你强大。是在等待中继续往前走，让你强大。

（金句收尾）

> 需卦说的不是"耐心等"。是"边等边走"。雨在天上一定会下来。你在雨来之前走的路——不会白走。等来坏消息，你已经做好了准备。等来好消息，你已经在路上了。两种结果，你都赢。
