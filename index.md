## 🧠 三大知识支柱（完美闭环）

```
        钱途心法（外层实操）
        /                    \
       /      完美闭环        \
      /                        \
   暗流规则（中层认知） ——— 觉醒/人性 (意识觉醒)
```

| 支柱 | 维度 | 核心命题 |
|------|------|---------|
| **钱途心法** | 外层实操 | 搞钱心法、人情世故、执行力、转运法则 |
| **暗流规则** | 中层认知 | 信息封锁、强势文化、人性秘术、权力结构 |
| **意识觉醒** | 内层意识 | NPC觉醒、系统设计、意识升级、维度思维 |

**三者闭环**：看见规则 → 觉醒意识 → 落地执行。缺一环都是空谈。
